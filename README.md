# 🌾 Exploring Agricultural Production Trends in Sri Lanka

## 📊 Project Overview
This project analyzes agricultural production trends in Sri Lanka to identify patterns, seasonal variations, and key insights that can support data-driven decision-making.

## 🎯 Objectives
- Analyze crop production trends over time
- Identify high-performing agricultural sectors
- Detect seasonal and regional variations
- Provide insights for policy and planning

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Power BI (if applicable)
- Excel

## 📂 Dataset
The dataset includes:
- Crop types
- Production quantities
- Time periods
- Regional data

## 📈 Key Insights
- Identified trends in major crop production
- Seasonal patterns affecting yield
- Regional differences in agricultural output
